1.0b19May 10, 2018 (Build 12976)

This release contains important changes, find out more about the latest updates in our blog post.
  • Controls can now be dragged inside a view in order to easily create subviews.
  • Introduces subnodes (accessible with a + button next to object).
  • Window exposed with new subnodes feature:
    • TabBar Item
    • NavigationBar Item
    • NavigationBar Left Item
    • NavigationBar Right Item
    • NavigationBar Back Item
  • TableView exposed with new subnodes feature:
    • Header
    • Footer
    • Cell
  • TableView Cell exposed with new subnodes feature:
    • Left Action
    • Right Action
    • Custom View
  • CollectionView exposed with new subnodes feature:
    • Cell
  • SegmentedControl exposed with new subnodes feature:
    • Item
  • Toolbar exposed with new subnodes feature:
    • System Item
    • Custom Item
  • TextView exposed with new subnodes feature:
    • Input View
    • Keyboard Accessory View
  • SearchBar exposed with new subnodes feature:
    • Keyboard Accessory View
  • TextField exposed with new subnodes feature:
    • Input View
    • Keyboard Accessory View
    • Left View
    • Righe View
  • Carousel exposed with new subnodes feature:
    • Cell
  • PageContainer exposed with new subnodes feature:
    • Cell
  • Removed fixed groups (Controls and Objects).
  • Added the ability to create folders in order to logically group controls and objects.
  • Project format has been internally changed.
  • Improved Date class.
  • Improved Data class.
  • Improved Color class.
  • Improved Console windows.
  • Improved Runtime Errors handling (a call stack is now logged).
  • Improved Simulator window.
  • Refactored Database and RecordSet classes.
  • All Database methods can now be executed sync or async (with closures).
  • RecordSet indexes are now all 0 bases.
  • Camera real-time preview in Design Board can now be disabled from within its inspector.
  • Added a new preference to force battery level to be 100% in Status Bar (when used in Mac laptops, useful for better screenshots).
  • Several fixes in the Inspector controls.
  • A more accurate float comparison has been introduced.
  • Tons of improvements in the Gravity bridge.
  • Tons of internal improvements to the main Creo app.
  • Tons of fixes and inconsistencies fixed.
  • CreoPlayer version 1.0.17 is now required
  • Improved documentation.
