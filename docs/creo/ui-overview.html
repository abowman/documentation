User Interface Overview

1   ACTIONS

Use to perform global actions for the current project.

  • Run project in Creo Simulator
  • Send project to a real device (via WiFi or USB)
  • Build application
  • Show 3D preview

2   INFO PANEL

Dynamically updated based on current container object. This section can also be used to select a different device and orientation.

  • Current container
  • Current device (editable)
  • Current orientation (editable)

3   WORKSPACE

Provides options to hide/change document sections.

  • Toggle code editor
  • Toggle animation editor
  • Toggle log panel
  • Toggle left pane
  • Toggle right pane

4   CONTAINERS PANEL

Provides shortcut for Layout objects creation.

  • Create a new Window
  • Create a new Navigation
  • Create a new Animation

5   DESIGN BOARD

Offers a visual representation of your app that can be easily modified with drag and drop.

  • Selected Window preview

6   INSPECTOR

Provides common configuration options for currently selected objects.

  • Object inspector
  • Control inspector
  • Actions
  • Bindings

7   OBJECTS and CONTROLS

Provides all the controls and objects that can be used in your app.

  • Controls
  • Non-UI Objects

8   PROJECT

Collect global resources available to all objects inside the app.

  • Application object
  • Assets
  • Globals
  • Templates

9   LAYOUT

Offers an app hierarchy overview.
